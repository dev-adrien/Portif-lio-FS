Conhecimentos em Programação e Versionamento

Durante minha formação, adquiri uma noção básica das linguagens Python, HTML e CSS, bem como do funcionamento do sistema de controle de versões Git. Esses conhecimentos têm sido fundamentais para compreender a lógica da programação, a estruturação de páginas web e a importância do versionamento no desenvolvimento de projetos colaborativos.

Em Python, compreendo conceitos como variáveis, estruturas condicionais e laços de repetição. Um exemplo simples que costumo praticar é o seguinte código que identifica se um número é par ou ímpar: 


            numero = int(input("Digite um número: "))
            if numero % 2 == 0:
            print("O número é par.")
            else:
            print("O número é ímpar.")

No desenvolvimento web, consigo estruturar páginas básicas utilizando HTML e aplicar estilos com CSS. Um exemplo simples de página com título centralizado e cor personalizada seria: 


            <!DOCTYPE html>
            <html>
            <head>
                <title>Minha Página</title>
                <style>
                    h1 {
                        color: steelblue;
                        text-align: center;
                    }
                </style>
            </head>
            <body>
                <h1>Bem-vindo à minha página!</h1>
            </body>
            </html>

Esse exemplo demonstra como criar uma página simples com um título centralizado e estilizado. A estrutura básica do HTML é composta por tags que definem o cabeçalho e o corpo da página, permitindo a inclusão de estilos CSS para personalização.

Além disso, entendo como funciona o Git, especialmente no que se refere aos comandos básicos de versionamento e colaboração, como git init, git add, git commit e git push. Essa ferramenta permite acompanhar a evolução de um projeto, revertendo alterações quando necessário e facilitando o trabalho em equipe.
